# pplan

商办写字楼租赁平台开发报价方案展示网站。

## 项目简介

本项目是一个面向商办写字楼租赁平台的定制开发报价方案展示系统，包含两套完整的开发方案：

- **方案一 (chooffice)**：以上海找办公室网为功能蓝本的开发报价方案
- **方案二 (soolou)**：以搜楼网为功能蓝本的开发报价方案

两个方案均参考 Industrious Office 的高端设计质量为视觉标准。

## 页面结构

```
├── index.html        # 方案导航页
├── chooffice.html    # 方案一详情页（蓝色主题）
└── soolou.html       # 方案二详情页（金色主题）
```

每个方案详情页包含以下模块：

| 序号 | 模块名称 | 说明 |
|------|----------|------|
| 00 | 项目概述 | 方案概览与核心指标 |
| 01 | 项目背景 | 项目背景与参考对标 |
| 02 | 功能模块 | 核心功能模块拆解 |
| 03 | 技术选型 | 技术栈与架构设计 |
| 04 | 团队配置 | 项目团队组成 |
| 05 | 开发排期 | 项目进度安排 |
| 06 | 报价明细 | 开发费用明细 |
| 07 | 运维保障 | 运维与售后方案 |
| 08 | 交付物 | 项目交付清单 |
| 09 | 付款方式 | 付款条款说明 |

## 技术栈

- **纯静态 HTML** - 无框架依赖，轻量高效
- **CSS3** - 自定义设计系统与动画效果
- **Google Fonts** - Playfair Display / Inter / Noto Serif SC / Noto Sans SC
- **Vercel** - 生产环境部署

## 设计系统

项目采用 CSS 变量驱动的设计系统，支持双主题配色：

| 主题 | 强调色 | 深底色 |
|------|--------|--------|
| 蓝色主题 (chooffice) | `#2E7DD1` | `#0F1923` |
| 金色主题 (soolou) | `#C8A96E` | `#1A1A2E` |

所有结构、排版、间距在 `styles.css` 中统一定义，页面仅通过内联 `:root{}` 覆盖颜色令牌即可切换主题。

## 部署

项目已部署至 Vercel，支持自动预览与生产构建：

```bash
# 开发预览
vercel

# 生产部署
vercel --prod
```
